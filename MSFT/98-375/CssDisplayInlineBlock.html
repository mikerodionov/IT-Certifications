Unordered List - display:list-item

list-item setting cause an element to be rendered like a block-level element, with addition of a bullet point preceding the content = default setting of li element

  • Apple
  • Banana
  • Strawberry

Unordered List - display:block

block setting cause an element to be preceded by a line break; the box of the element itself will take all of the available width

  • Apple
  • Banana
  • Strawberry

Unordered List - display:inline-block

inline-block setting cause an element to appear horizontally adjacent to other elements that fit on the same line

  • Apple
  • Banana
  • Strawberry

Unordered List - display:inline

inline setting cause an element to be broken into one or more boxes; box will take the remaining width of the current line or the exact width required for the content, whichever is lesser, additional boxes will be rendered on subsequent lines

  • Apple
  • Banana
  • Strawberry
